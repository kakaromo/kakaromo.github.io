## 3층 구조

- **L1 (전체상)**: 시스템 블록 다이어그램. 30분 이내에 "이 프로젝트가 무엇을 하는가"를 답할 수 있게 하는 것이 목표.
- **L2 (기능 모듈)**: 한 기능을 데이터 흐름 순서로 따라갑니다. 현재는 슬롯 모니터링(Head TCP + SSE)만 완성되어 있고, 같은 템플릿으로 벤치마크·원격·Agent 등이 추가될 예정.
- **L3 (언어 개념)**: L2 본문에 등장하는 어노테이션·rune을 호버하면 요약이 뜨고, 클릭하면 해당 개념의 상세 페이지로 이동합니다.

## 시각화 요소

학습 페이지에서 다음 상호작용 요소를 만날 수 있습니다.

- **아키텍처 다이어그램**: 노드에 마우스를 올리면 강조되고, 클릭하면 해당 코드 또는 다음 단계로 이동합니다.
- **코드 투어**: 왼쪽 설명을 스크롤하면 오른쪽 코드가 해당 부분으로 이동·강조됩니다.
- **시퀀스 타임라인**: TCP 메시지가 SSE를 거쳐 브라우저까지 도달하는 과정을 재생해 볼 수 있습니다. `prefers-reduced-motion`을 켠 경우 자동 재생은 꺼집니다.
- **용어 팝오버**: 밑줄 있는 용어를 호버(또는 Tab + Enter)하면 요약이 뜹니다.
- **Before/After 슬라이더**: 리팩토링 Phase별 코드 변화를 좌우로 비교합니다.

## "직접 해보기" 박스

각 L2 페이지 하단에는 주황색 레일이 붙은 **직접 해보기** 박스가 있습니다. 이 박스만 순서대로 따라가면 단일 호환성 Head를 연결해 브라우저에서 슬롯 상태를 받는 것까지 end-to-end로 실행할 수 있습니다. 읽기만 할 경우 건너뛰어도 무방합니다.

## 접근성

- 모든 인터랙티브 요소는 키보드(Tab / Shift+Tab / Enter / Esc / 화살표)로 조작할 수 있습니다.
- 다이어그램·타임라인은 항상 텍스트 대체(접기/펼치기 설명)를 함께 제공합니다.
