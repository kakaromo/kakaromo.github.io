## 이 가이드는 누구를 위한 것인가

- Samsung 내부의 신규 개발자
- Java/Spring 경험은 있지만 MOVE 포털 또는 Svelte 5가 처음인 분
- 한 번 작업했지만 몇 달 뒤 돌아와 다시 맥락을 맞춰야 하는 분

언어 기초 전반을 다루지 않습니다. **이 프로젝트에서 실제 쓰인 것만** 설명합니다. 더 깊게 들어가고 싶을 때는 각 개념 페이지 하단의 공식 문서 링크를 따라가세요.

## 구조

- **L1 — 프로젝트 전체상**: 도메인, 시스템 아키텍처, 기술 스택, 핵심 데이터 흐름 3선
- **L2 — 기능 모듈**: 슬롯 모니터링부터. 다른 기능은 같은 템플릿으로 점진 추가
- **L3 — 언어/프레임워크 개념**: L2 페이지에서 등장하는 용어의 호버 팝오버가 연결되는 참조 사전

## L2를 여러 개 읽은 뒤에는

네 모듈을 한 번씩 훑었다면 [**L2 4종 비교**](/learn/l2-compare/)에서 공통 패턴과 갈림길을 한 페이지로 확인하세요. 다섯 번째 L2를 쓰려는 경우 체크리스트로도 쓸 수 있습니다.
