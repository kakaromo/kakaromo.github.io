성능 테스트 결과를 Excel 파일로 내보내는 기능입니다. 프론트엔드에서 직접 생성하는 방법과 백엔드(Go Excel Service)를 통해 네이티브 Excel 차트가 포함된 파일을 생성하는 방법, 두 가지를 제공합니다.

## 방법 1: 프론트엔드 Excel Export

각 성능 차트 컴포넌트에 포함된 **Excel** 버튼을 클릭하여 현재 보고 있는 차트 데이터를 Excel 파일로 다운로드합니다.

### 사용 방법

1. Performance History 상세 페이지에서 차트를 확인합니다.
2. 차트 상단의 **Excel** 버튼을 클릭합니다.
3. 브라우저에서 `.xlsx` 파일이 다운로드됩니다.

:::note
프론트엔드 Export는 데이터만 포함되며, Excel 차트(그래프)는 포함되지 않습니다.
:::

## 방법 2: 백엔드 Excel Export (네이티브 차트 포함)

Go Excel Service를 통해 생성되는 Excel 파일에는 **네이티브 Excel 차트**가 포함됩니다. Excel에서 직접 차트를 확인하고 편집할 수 있어 보고서 작성에 적합합니다.

### 사용 방법

1. Performance History 목록 페이지로 이동합니다.
2. 원하는 History 행의 **다운로드** 버튼을 클릭합니다.
3. 서버에서 Excel 파일이 생성되며, 완료 후 자동으로 다운로드됩니다.

:::tip
백엔드 Export는 데이터 양이 많을 경우 생성에 수 초가 소요될 수 있습니다. 다운로드 버튼 클릭 후 잠시 기다려주세요.
:::

## 두 방법 비교

| 구분 | 프론트엔드 Export | 백엔드 Export |
|------|-------------------|---------------|
| 차트 포함 | 데이터만 | 네이티브 Excel 차트 포함 |
| 생성 위치 | 브라우저 | 서버 (Go Excel Service) |
| 속도 | 빠름 | 데이터 양에 따라 다소 소요 |
| 사용처 | 빠른 데이터 확인 | 보고서/공유용 |

## 지원되는 파서 목록

백엔드 Excel Export는 다음 15종의 파서를 지원합니다. 각 파서에 맞는 전용 차트가 Excel 파일에 생성됩니다.

- GenPerf
- FragmentWrite
- WearLeveling
- 그 외 12종의 파서별 전용 차트

:::note
지원되지 않는 parserId의 History는 백엔드 Export 시 데이터만 포함됩니다.
:::
