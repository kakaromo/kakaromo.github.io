대시보드는 Portal에 접속하면 가장 먼저 보이는 화면입니다. 테스트 현황을 한눈에 파악할 수 있도록 통계 카드, 차트, 요약 테이블, 실시간 슬롯 상태를 제공합니다.

## 통계 카드

화면 상단에 호환성/성능 테스트의 주요 수치가 카드 형태로 표시됩니다.

| 카드 | 설명 |
|------|------|
| **TR 수** | 등록된 Test Request 총 개수 |
| **History 수** | 실행된 테스트 History 총 개수 |
| **Pass Rate** | 전체 테스트 중 PASS 비율 (%) |

호환성과 성능 테스트 각각의 카운트가 별도로 표시됩니다.

## DonutChart — 결과 분포

테스트 결과를 **PASS / FAIL / OTHER** 세 가지 범주로 분류하여 도넛 차트로 시각화합니다. 각 영역에 마우스를 올리면 정확한 수치와 비율을 확인할 수 있습니다.

## BarChart — FW별 통계

테스트가 가장 많이 실행된 상위 10개 FW(Firmware)의 통계를 막대 차트로 표시합니다. FW별 PASS/FAIL 비율을 직관적으로 비교할 수 있습니다.

## Summary 테이블

FW별, TC별로 세분화된 pass/fail/rate 수치를 테이블로 제공합니다. 특정 FW나 TC의 성능 추이를 빠르게 확인할 때 유용합니다.

## 최근 10건 History

가장 최근에 실행된 10건의 테스트 History가 목록으로 표시됩니다. 각 항목을 클릭하면 해당 History의 상세 결과 페이지로 이동합니다.

## 실시간 슬롯 요약 타일

대시보드 하단에는 현재 슬롯 상태가 타일 형태로 요약 표시됩니다. SSE(Server-Sent Events)를 통해 실시간으로 업데이트되므로 페이지를 새로고침하지 않아도 최신 상태를 확인할 수 있습니다.

:::tip
슬롯의 상세 상태와 제어 기능은 [실시간 슬롯 모니터링](/guide/slot-monitoring/) 페이지에서 확인하세요.
:::

## API

대시보드 데이터는 `GET /api/dashboard/stats` API를 통해 조회됩니다. 페이지 진입 시 자동으로 호출되며, 별도의 설정 없이 최신 통계가 표시됩니다.
