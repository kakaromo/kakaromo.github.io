TC 그룹은 자주 사용하는 Test Case 조합을 미리 저장해두고, 슬롯에 TC를 할당할 때 한 번의 클릭으로 일괄 선택할 수 있는 기능입니다.

## TC 그룹이란

테스트를 반복 실행할 때마다 동일한 TC를 하나씩 선택하는 것은 번거롭습니다. TC 그룹을 사용하면:

- 자주 사용하는 TC 조합을 이름을 붙여 저장
- 슬롯 페이지에서 Chip 한 번 클릭으로 전체 TC 일괄 선택
- 팀 내에서 표준 테스트 세트를 공유

## 그룹 생성

1. TC 그룹 관리 페이지에서 **[+ Save Group]** 버튼을 클릭합니다.
2. 다이얼로그가 열리면 그룹 이름을 입력합니다.
3. TC 목록에서 포함할 TC를 선택합니다.
4. **저장** 버튼을 클릭하여 그룹을 생성합니다.

:::tip
그룹 이름은 용도를 명확히 알 수 있도록 지정하세요. 예: `GenPerf-기본세트`, `Aging-풀테스트`
:::

## 그룹 사용 — Slots 페이지에서 빠른 적용

TC 그룹의 핵심 사용처는 **Slots 페이지의 SetTC Sheet**입니다.

1. Slots 페이지에서 슬롯을 선택합니다.
2. Context Menu 또는 Selection Sheet에서 **Set TC**를 클릭합니다.
3. SetTC Sheet 상단에 저장된 TC 그룹이 **Chip** 형태로 표시됩니다.
4. 원하는 그룹 Chip을 클릭하면 해당 그룹에 포함된 TC가 자동으로 선택됩니다.

## 그룹 관리

### 수정

기존 그룹의 이름이나 포함 TC를 변경할 수 있습니다. 그룹 목록에서 편집 버튼을 클릭하여 다이얼로그에서 수정합니다.

### 삭제

더 이상 필요 없는 그룹을 삭제합니다. 삭제해도 이미 슬롯에 할당된 TC에는 영향이 없습니다.

:::caution
그룹 삭제는 되돌릴 수 없습니다. 삭제 전에 확인 다이얼로그가 표시됩니다.
:::

## 호환성/성능 TC 별도 관리

호환성 테스트 TC와 성능 테스트 TC는 별도의 그룹으로 관리됩니다. 각각의 테스트 유형에 맞는 TC만 그룹에 포함할 수 있습니다.
