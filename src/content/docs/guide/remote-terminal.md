---
title: 원격 터미널 (SSH/RDP)
description: 웹 브라우저에서 SSH/RDP로 테스트 VM에 원격 접속하는 방법
---

원격 터미널은 웹 브라우저에서 별도의 SSH 클라이언트 없이 테스트 VM에 직접 접속할 수 있는 기능입니다. 페이지 경로: `/remote`

## 개요

xterm.js 기반 SSH 터미널로 브라우저에서 직접 VM에 접속합니다. VPN이나 별도 소프트웨어 설치 없이 웹 브라우저만으로 원격 접속이 가능합니다.

- **텍스트 선택**: 마우스 드래그로 자연스럽게 선택
- **복사**: Cmd+C (선택 텍스트가 있으면 복사, 없으면 SIGINT)
- **붙여넣기**: Cmd+V (브라우저 네이티브)

## VM 선택

접속할 VM을 선택합니다. 사용 가능한 VM 목록이 표시됩니다:

- **T1 ~ T4** — Tentacle 서버 (테스트 실행 환경)
- **HEAD** — Head 서버 (중앙 제어)

## 다중 터미널 탭

여러 VM에 동시에 접속할 수 있습니다. 각 접속은 별도의 탭으로 관리됩니다.

- 탭 클릭으로 터미널 간 전환
- 탭의 X 버튼으로 접속 종료
- 필요한 만큼 새 탭 추가 가능

## Broadcast 명령어

**Broadcast** 기능을 사용하면 열려 있는 **모든 터미널 탭에 동시에 동일한 명령어**를 전송할 수 있습니다.

여러 VM에 같은 명령을 실행해야 할 때 유용합니다. 예를 들어:

- 모든 Tentacle에서 로그 확인
- 일괄 서비스 재시작
- 동시 상태 점검

:::tip
Broadcast는 SSH 터미널에서만 동작합니다. RDP 탭에는 영향이 없습니다.
:::

## 슬롯 페이지에서 터미널 열기

Slots 페이지에서 슬롯을 우클릭한 후 **Terminal** 메뉴를 선택하면, 해당 슬롯이 연결된 VM으로 바로 원격 터미널이 열립니다. 별도로 VM을 선택할 필요가 없습니다.

## 전체화면 토글

터미널 영역을 전체화면으로 확장할 수 있습니다. 전체화면 버튼을 클릭하거나 단축키를 사용합니다. 다시 클릭하면 원래 크기로 복원됩니다.
